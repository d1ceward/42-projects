# 42-libft
The libft is a C languague library (like libc with less & more functions) for everyday use at school 42.
## Convert functions
```C
int		ft_abs(int n);
int		ft_atoi(const char *nptr);
char	*ft_itoa(int n);
int		ft_tolower(int c);
int		ft_toupper(int c);
```
