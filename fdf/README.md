# 42-fdf
The goal of this project was create graphical schematic representation of embossed land by connecting different points (x, y, z) by straight lines.

## Launch

```Shell
./fdf <filename>
```

Control
---
#####Rotation#####
	A <-> D  =>  Y Axis
	W <-> S  =>  X Axis
	Q <-> E  =>  Z Axis
#####Translation#####
	Z <-> X  =>  Horizontal
	R <-> F  =>  Vertical
#####Zoom#####
	T <-> G  =>  Zoom
